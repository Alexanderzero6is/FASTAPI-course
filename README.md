## Configurar el entorno en ArchLinux
Tener instalado el Vscode, de lo contrario instalarlo [Vscode page](https://code.visualstudio.com/)
Tener instalado python, de lo contrario instarlo [Python page](https://www.python.org/)

Comprobar la versión de python de su ordenador, se comprueba de la siguiente manera en la terminal:
```
python3 -V 
```

### Crear el entorno virtual y seleccionarlo
Abrimos la terminal de Vscode y escribimos:
```
python3 -m venv [nombre del entorno virtual]
```
Después apretamos la combinación Ctrl+Shift+P y escribimos:
```
Python: Select Interpreter
```
y seleccionamos nuestro entorno virtual

Activamos tu entorco virtual escribiendo en la terminal:
```
source venv/bin/activate
```
